Agora para configurar Groq:
1. Reinicie o Windsurf (fechar e abrir novamente)
2. Clique no ícone do Cline na barra lateral
3. Clique no ícone de engrenagem (settings)
4. Em API Provider → selecione "Groq"
5. Em Groq API Key → cole sua key (pega aqui (https://console.groq.com/keys))
6. Escolha o modelo: qwen-qwq-32b ou llama-3.3-70b
Você tem a API key da Groq? Se não, cria uma lá e me diz que te ajudo.
